# Learning Resources & Deep Dive Guide

Curated paths for going from practitioner to expert in ML/AI engineering.

## Learning Roadmaps

### Path 1: ML Foundations ‚Üí Production
```
1. Python + NumPy/Pandas (2 weeks)
2. ML Fundamentals with sklearn (2 weeks)
3. Deep Learning with PyTorch (4 weeks)
4. Transformers & NLP (2 weeks)
5. MLOps & Production (2 weeks)
```

### Path 2: Software Engineer ‚Üí ML Engineer
```
1. ML Math Refresher (linear algebra, calculus, probability)
2. Classical ML (sklearn)
3. Neural Networks from Scratch
4. PyTorch Deep Dive
5. Specialized Domain (NLP/CV/RL)
```

### Path 3: ML Engineer ‚Üí AI Infrastructure
```
1. CUDA Programming Fundamentals
2. GPU Architecture Deep Dive
3. Inference Optimization (TensorRT, quantization)
4. Distributed Systems for ML
5. MLOps at Scale
```

## Key Papers to Read

  ‚Üí https://arxiv.org/abs/1706.03762

  ‚Üí https://arxiv.org/abs/2005.14165

  ‚Üí https://arxiv.org/abs/1512.03385

  ‚Üí https://arxiv.org/abs/1502.03167
